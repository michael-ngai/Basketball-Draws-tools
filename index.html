Draw rules
Games per week

Each division plays every Friday. The number of games per week depends on team count:

  • Even number of teams: games = total teams ÷ 2
  • Odd number of teams: games = (total teams + 1) ÷ 2

Current: U10 = 2 games, U12 = 2 games, U14 = 3 games per week.

Courts

There are 2 courts — Court 1 and Court 2. Both can run games at the same time slot.

No double-booking

No team can play 2 games at the same time slot. This is a hard rule and cannot be overridden.

Fill-in team (Fill-in)

When a division has an odd number of teams, one team per week plays an extra game as a fill-in. For example with teams A, B, C:

  • Round robin game: A vs B
  • C would have a bye — to keep it fair, A or B plays C as a fill-in
  • The fill-in team's extra game only is labelled (Fill-in). Their round robin game has no label.

Max appearances per team per week: 2 (once in RR, once as fill-in).

Round robin order

Round robin games are scheduled before fill-in games each week. The tool tries to avoid repeating the same matchup 2 weeks in a row. If unavoidable due to availability, it allows it but compensates in future weeks so each team plays each other an equal number of times overall. This is a soft rule.

Scheduling priority

When multiple games compete for the same slot, the tool prioritises:

  1. Most constrained matchup first (fewest shared available time slots)
  2. If equal constraint — youngest division first: U10 → U12 → U14
Back-to-back games (no gaps)

All games should be scheduled consecutively with no empty slots in between. This keeps referees busy without waiting. The tool generates:

  • Option 1 (preferred): No gaps — all games back-to-back
  • Option 2+: Has a gap — flagged with ⚠️ warning. Requires your approval before saving.
Start time

The tool prefers to start games at 4:45pm or 5:30pm. If no teams are available at those times, it will use 6:15pm or 7:00pm freely — any slot is valid as long as the team is available.

Same-club scheduling Hard rule (except GSA)

Teams from the same club (e.g. Silverfang U10, Silverfang U12) must never be scheduled at the same time slot on different courts — a coach cannot be in two places at once.

  • No same-time clash: Hard rule — the tool will never place two same-club games at the same time. If it cannot avoid it (no other slot works), it flags a conflict for you to fix manually.
  • Games in a row: Soft rule — same-club games are scheduled consecutively where possible so the coach doesn't wait. A gap is flagged with ℹ️ if unavoidable.

GSA is exempt from both rules (they have multiple coaches). Club names are detected from the team name pattern: Age Group → Club → Team Name.

Options shown

The tool always generates multiple draw options for you to choose from. If all options have issues, all are shown with conflicts explained so you can pick the best one and fix it manually in Season Tracker.

Season rules
Season dates

Season runs 1 May 2026 – 14 August 2026, every Friday. No games on 17 July and 24 July (winter school holidays).

Regular rounds

Weeks 1–12 are regular rounds.

Week 13 — Semi finals
  • 3 teams or less: No semi finals. Top 2 go straight to grand final in week 14.
  • 4 teams or more: Top 4 play semi finals (1st vs 4th, 2nd vs 3rd).
  • Teams that don't make semis still play a makeup game.
Week 14 — Grand finals

All divisions play their grand final. Teams that don't make the final play a makeup game.

Finals game ordering

In weeks 13 and 14, if a team plays both a finals game and a makeup game, the finals game is scheduled first (earlier slot). This lets players put their best effort into the important game while fresh. The makeup game goes second. If this isn't possible due to availability, it's flagged.
